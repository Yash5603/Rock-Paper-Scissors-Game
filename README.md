# 🎮 Rock-Paper-Scissors Game 🤖

This is an interactive Rock Paper Scissors game developed using HTML, CSS, and JavaScript. The game allows a player to compete against the computer in real time. The computer automatically generates a random choice (Rock, Paper, or Scissors), and the game instantly compares both selections to determine the winner.

# Features :

<li>🎮 Player vs Computer Gameplay</li>
<li>🤖 Random computer choice generation</li>
<li>📊 Live score tracking with unlimited rounds</li>
<li>🏆 Displays Win, Lose, or Draw result messages after every round</li>
<li>🔄 Play continuously without restarting the game</li>
<li>⚡ Instant result calculation using JavaScript</li>
<li>🎨 Responsive and user-friendly interface designed with CSS</li>
