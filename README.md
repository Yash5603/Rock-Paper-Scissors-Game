# 🎮 Rock-Paper-Scissors Game 🤖

This is an interactive Rock Paper Scissors game developed using HTML, CSS, and JavaScript. The game allows a player to compete against the computer in real time. The computer automatically generates a random choice (Rock, Paper, or Scissors), and the game instantly compares both selections to determine the winner.

# Features:
🎮 Player vs Computer Gameplay
🤖 Random computer choice generation
📊 Live score tracking with unlimited rounds
🏆 Displays Win, Lose, or Draw result messages after every round
🔄 Play continuously without restarting the game
⚡ Instant result calculation using JavaScript
🎨 Responsive and user-friendly interface designed with CSS
